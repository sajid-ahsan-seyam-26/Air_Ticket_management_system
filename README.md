# ✈️ Air Ticket Management System

A simple Air Ticket Management System built using Python (Tkinter GUI).
This project allows users to book, search, update, and cancel flight tickets easily.

# 📌 Features
✅ Add Ticket Booking
✅ View All Tickets
✅ Search Ticket by Booking ID
✅ Search Ticket by Passport Number
✅ Update Ticket Information
✅ Cancel Ticket
✅ Show Available Seats
✅ Show Total Sales
✅ Simple GUI using Tkinter
🛠️ Technologies Used
Python 🐍
Tkinter (GUI Library)
# 🚀 How to Run
Install Python (if not installed)

Clone the repository:

# git clone https://github.com/sajid-ahsan-seyam-26/Air_Ticket_management_system.git

Go to the project folder:

# Run the program:

python main.py

# 🧠 How It Works
User inputs passenger and flight details
System calculates total fare
Data is stored in a list (temporary storage)
Users can perform CRUD operations:
Create → Add Ticket
Read → Show/Search Ticket
Update → Modify Ticket
Delete → Cancel Ticket
# ⚠️ Limitations
❌ No database (data lost after closing app)
❌ No login system
❌ Seat validation is basic
